## 誕生の経緯

やあ、これは全く新しいリーチ麻雀教育ソフトです。

ちょっと長い話になりますが——ずっと、本当に麻雀を教えてくれるツールが欲しかったんです。でも市場にある既存のソフトやモデルを片っ端から試しても、どうもしっくり来ない。牌効率計算機（mahjong-helper 系）は向聴数・有効牌・打点といった確定的なデータは計算できるけど、それだけ——数字は分かっても、全体の方針が見えてこない。モデルローダー（Akagi 系）は裏で Mortal などのニューラルネットワークを動かしていて確かに強い、でも完全にブラックボックス——推奨結果は見えるのに、モデルが何を「考えている」のか全く分からない。まるで答えを丸写しして、結局何も理解できていない感じです。

僕が求めていたのは、「この牌を推奨する理由」まで教えてくれる AI。しかもその「理由」は後付けの結果論じゃなくて、推論時にモデルが**直接出力する**内在的な判断であること。たとえば今の局面がどれくらい危険か、押す自信がどの程度あるか、この一手は攻めなのか守りなのか——これらはモデル自身が「語る」もので、ルールから逆算して取り繕ったものじゃありません。

というわけで、会社から仲間を何人か「拉致」して、一緒にニューラルネットワークという沼に飛び込みました。もう戻れません（笑）

モデルについて少し。FlyA は純粋な人間の棋譜を使った行動模倣を基盤に訓練しています。アーキテクチャは CNN + Transformer で、注意機構・長期時系列認識・相手の推測能力を備えています。さらに CFR（反実仮想後悔最小化）アルゴリズムによる行動最適化と、自己対局による強化学習を組み合わせ、訓練は複数段階で進めています。まだ初期段階ですが、ベースモデルだけで既に RiichiLab で1位を獲得しました。

それから、ネットワークアーキテクチャにかなり……ぶっ飛んだ最適化を施した結果、ごく少量の牌譜から深い麻雀戦略を抽出できるようになりました。なので、ユーザーの対局データを収集する心配はご無用です——ぶっちゃけ、そんなに大量の牌譜は必要ないので :)

Discord / QQ グループに参加して、モデル訓練の進捗やソフトリリース情報をチェックしてください——無料キーをばら撒きますよ！太っ腹でいきます！

## 概要

**FlyA Agent** は FlyA AI を搭載した日本リーチ麻雀 AI デスクトップソフトウェアです。リアルタイム教育、内蔵対局、自動化操作を一つに統合しています。

雀魂などのリーチ麻雀ゲームに接続して、AI のリアルタイム推奨と思考過程を見ることができます。内蔵の麻雀ゲームで AI と直接対局することも可能です。FlyA モデルの最大の特徴は**説明可能性**——脅威認識、推進自信度、意思決定の動機は、すべてモデルの推論プロセスで直接出力されるシグナルであり、後から付け足した分析ではありません。

> **現在のバージョンは早期テスト版です。** バグは確実にあります。気になる方は、ガチで段位を上げたいメインアカウントでの使用はお控えください :)

## ダウンロード

[Releases](../../releases/latest) ページから最新版をダウンロードしてください：

| ファイル | 説明 |
|----------|------|
| `FlyA-Agent-*-win-x64-setup.exe` | Windows インストーラー（推奨） |
| `FlyA-Agent-*-win-x64.zip` | Windows ポータブル版（インストール不要） |

**動作環境**：Windows 10/11 (x64)、インターネット接続必須（AI 推論はクラウドサービス）

Go と Rust で書かれた静的コンパイル済みソフトで、ランタイム依存ゼロ。インストールしたらすぐ使えます。もし何か追加インストールが必要になったら、それは完全にこちらのバグです。教えてください。

## クイックスタート

1. インストールまたは解凍後、**FlyA Agent** を起動
2. テストキーでログイン（下記参照）
3. ホーム画面の「クイックスタート」カードからゲームプラットフォームを選択：
   - **雀魂 Web 版**：内蔵フィンガープリントブラウザで起動、ブラウザ指紋は漏れません
   - **クライアント版**（雀魂クライアント、一番街など）：「プロキシ開始」をクリックして管理者権限を付与すると、証明書と仮想ネットワークアダプタの設定が自動完了
   - **手動で証明書インストール + 自前のプロキシ**を使う方法も可能（管理者権限不要）

## モデルとログイン

テスト版は 2 つのログイン方法に対応：
- **Akagi OT2 Key**
- 不定期に配布する**テストキー**

補足ですが、説明可能性（脅威認識・推進自信度・意思決定動機）は FlyA 独自モデルの推論時に直接出力される結果であり、ルールベースの後付け分析ではありません。OT モデルではこの機能は使えませんが、向聴数・安全牌・有効牌などの基本的な確定計算は利用可能です。

## プロキシと権限

クライアントプロキシモードは独自開発のプロキシエンジンを使用し、仮想ネットワークアダプタでゲームトラフィックをルーティングします。初回使用時に管理者権限が必要です：

- 信頼済み証明書のインストール
- 仮想ネットワークアダプタの作成

一度設定すれば、以降はバックグラウンドで静かに動作します。心配な方は、手動で証明書をインストールして自前のプロキシツールを使うこともできます。

> ⚠️ TUN モードを使用するため、Clash や V2RayN など同じく TUN モードのプロキシソフトとは共存できません。事前にそれらを終了するか、システムプロキシモードに切り替えてください。

## HUD について

HUD オーバーレイはデフォルトでスクリーンショットに映りません（配信・録画時の漏洩防止）。バグ報告用にスクリーンショットが必要な場合は、設定 > HUD からスクリーンショット表示を有効にしてください。

## プライバシーとセキュリティ

- **データ安全性**：ソフトウェアはゲームデータを改ざんしたり、個人情報をこっそりアップロードしたりしません。モデル訓練にプレイヤーデータも必要ありません。
- **BAN リスク**：テストアカウントは今のところ全員無事です。ただし、長時間の非人間的な自動操作は BAN される可能性があります——ご利用は計画的に。
- **クローズドソース**：悪用防止とユーザー保護のため、長期的にクローズドソースとし、大規模な無料公開は行いません。完全自社開発で、同種ソフトのコードは一切使用していません。まあ、ソフト自体がコア資産というわけでもないので、性能を犠牲にするような重いプロテクトをかける気もないですけどね。
- **ウイルス対策ソフト**：現行バージョンは標準ビルドパイプラインを使用しており、誤検知はかなり減っています。それでも検知された場合はご連絡ください。

## アップデートについて

正直なところ、初期バージョンの今後の更新頻度は低くなる見込みです。モデルアーキテクチャが大きく変わっており、現在ソフトウェアは「ブリッジ」方式でモデルと通信しています。エンジン交換はパイプライン全体の書き直しに等しいので、計画としては：まず使えるバージョンをリリース → 更新を一時停止 → 新モデル対応に全集中、という流れです。
