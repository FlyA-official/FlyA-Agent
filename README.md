# FlyA Agent

**FlyA Agent** 是一款基于 FlyA AI 的日本麻将智能体桌面软件。

## 功能特性

### FlyA Mahjong（游戏模块）
- 内置对局系统（1 人类 + 3 AI 实时对局）
- 支持四麻 / 三麻
- 完整日麻规则引擎

### FlyA Coach（辅助模块）
- FlyA AI 实时打牌推荐
- 支持雀魂、一番街、天凤、雀姬
- HUD 浮窗 + WebUI 推荐展示
- 自动化操作

## 系统要求

- Windows 10/11 (x64)
- 需要联网（AI 推理为云端服务）

## 快速开始

1. 下载并安装 / 解压
2. 启动 FlyA Agent
3. 登录账号
4. 开启游戏平台，FlyA 自动连接并提供推荐
